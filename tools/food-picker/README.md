# 苏州今天吃什么

一个本地运行的苏州餐厅选择器，支持多维度筛选和随机推荐。不联网，数据全部内嵌。

## 运行

```bash
python suzhou_food_picker.py
```

## 功能

- **多维筛选**：类型、区域、预算、时间段、场景、口味、关键词
- **随机推荐**：带抽签动画的随机选择
- **备选项**：推荐后显示 5 个备选
- **历史记录**：记住最近 8 次推荐结果
- **一键复制**：复制结果可直接发给朋友

## 自定义

想添加或修改餐厅，直接编辑 `suzhou_food_picker.py` 中的 `RESTAURANTS` 列表即可。

## 注意

- 本软件不联网
- 餐厅营业时间、分店位置请出发前用地图确认
